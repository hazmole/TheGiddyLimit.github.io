有在用Discord嗎？

你可以加入5etool開發團隊的伺服器，查看最新的公告、更新項目、以及參與功能投票。

有在用Foundry嗎？

沒有？那麼你應該試試！ Foundry是一個現代化、優先支援mod修改功能，比Roll20還要更好的替代品。它是5etool整合的首選平台。因此，5etool開發團隊將不會再為betteR20進行基本功能維護以外的更新。相對的，開發團隊將把心力轉向為Foundry創建一套擴展工具集，並以Plutonium模組的形式發佈。

若你想要安裝這個模組，複製清單URL，將之貼到Foundry的Install Module對話框中。參閱Plutonium Features頁面以瞭解關於該模組所能提供的功能。

有在用Roll20嗎?

Please note: the 5etools Roll20 integration, betteR20, is no longer being actively developed. Instead, development efforts are being focused on Plutonium, as described above.

Install Roll20 Enhancement Suite and Tampermonkey (Chrome/Firefox), and get the betteR20 script using this link.

As a player, you can import character options, spells, and items to quickly fill out a sheet. As a GM, you can additionally import monsters, tables, and adventures.

To import monsters, your game must have a default character sheet. We strongly recommend using OGL when importing, then switching to another afterwards if desired. (When switching to the Shaped sheet, be sure to open up NPC sheets after import to let them convert.)

Check the installation guide on the wiki, and check the Help page for some basic pointers on how to use the script.

FAQ

這個網站有相關的wiki嗎？ 在這。 歡迎貢獻一份心力。

這是免費的嗎？ 是的！另外，你還可以在get.5e.tools下載這個網站的原始碼。參見README或wiki以查閱更多詳情。

我可以為這個網站提供貢獻嗎？ 你可以在GitHub上取得這個網站的原始碼副本。要注意的是，這僅是一份輔助鏡像資料，因此並不能保證它是最新的版本。

瀏覽器和裝置支援？ 最新的桌面板 Chrome 和 Firefox。

我的東西都放在哪裡？ 這個網站並沒有使用帳號系統。你所有的設定都存在cookies。如果你把它們清除、或是關掉你的無痕視窗，那麼它們就會永遠消失。你可以透過設定功能表下載你的資料，以便妥善保存。

還有什麼是我需要知道的嗎？ 你可以在懸停視窗出現時按住SHIFT來把該視窗固定。這個網站的幾乎所有東西都有使用提示。然後……你永遠都會是個DM。
